# LinGuinE
LongitudINal GUIdance Estimation

LinGuinE is a framework for segmenting volumetric medical images longitudinally developed at AstraZeneca. For more information about the method, please see our preprint: https://arxiv.org/abs/2506.06092.

## Release Timeline

2. Code release - estimated 15/05/2026
